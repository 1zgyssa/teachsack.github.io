# 教学工具箱浅色/深色模式切换方案

## 背景

用户希望为教学工具箱添加浅色/深色模式切换功能，提升用户体验。需要在不影响现有功能（点名器、刮刮乐、随机分组、成绩统计）的前提下，实现主题切换。

## 需求确认

* 仅支持浅色/深色两种模式（不跟随系统）

* 使用 🌙/☀️ 简单图标

* 切换时添加平滑过渡动画

* 确保所有功能适配两种主题

## 实施方案

### 1. CSS变量定义

在 `<style>` 标签开头添加主题变量，使用 `[data-theme]` 属性选择器：

```css
:root {
    --bg-primary: #f5f7fa;
    --bg-secondary: #ffffff;
    --bg-gradient-start: #e8ecf1;
    --bg-gradient-end: #d1d8e3;
    --text-primary: #1a1a2e;
    --text-secondary: #5a6677;
    --accent-primary: #f0a500;
    --card-bg: rgba(255, 255, 255, 0.7);
    --border-color: rgba(0, 0, 0, 0.08);
    /* ... 其他变量 */
}

[data-theme="dark"] {
    --bg-primary: #0d0d1a;
    --bg-gradient-start: #1a1a2e;
    --bg-gradient-end: #0d0d1a;
    --text-primary: #ffffff;
    --text-secondary: rgba(255, 255, 255, 0.8);
    --accent-primary: #ffd700;
    --card-bg: rgba(255, 255, 255, 0.05);
    --border-color: rgba(255, 255, 255, 0.08);
    /* ... 其他变量 */
}
```

### 2. 添加过渡动画

```css
body, .app, .tool-card, .content-card, button, input, textarea {
    transition: background-color 0.3s ease, color 0.3s ease, border-color 0.3s ease;
}
```

### 3. 切换按钮HTML

在顶部工具栏添加按钮（建议在全屏按钮左侧）：

```html
<button class="theme-btn" id="themeBtn" onclick="toggleTheme()" title="切换主题">🌙</button>
```

### 4. JavaScript逻辑

```javascript
const ThemeConfig = {
    STORAGE_KEY: 'teaching-toolbox-theme',

    init() {
        const saved = localStorage.getItem(this.STORAGE_KEY);
        this.applyTheme(saved || 'dark');
    },

    applyTheme(theme) {
        document.documentElement.setAttribute('data-theme', theme);
        const btn = document.getElementById('themeBtn');
        if (btn) btn.textContent = theme === 'dark' ? '☀️' : '🌙';
    },

    toggle() {
        const current = document.documentElement.getAttribute('data-theme') || 'dark';
        const next = current === 'dark' ? 'light' : 'dark';
        localStorage.setItem(this.STORAGE_KEY, next);
        this.applyTheme(next);
    }
};

function toggleTheme() { ThemeConfig.toggle(); }
document.addEventListener('DOMContentLoaded', () => ThemeConfig.init());
```

### 5. 需要修改的关键颜色（高频元素优先）

* 背景渐变：body, .app

* 文字颜色：h1, h2, h3, p, .hint-text

* 卡片背景：.tool-card, .content-card, .import-area

* 按钮样式：.ctrl-btn, .theme-btn

* 输入框：input, textarea

* 边框颜色：.card-border, button border

**保持硬编码的元素**（特殊视觉效果）：

* 卡牌背面颜色（扑克牌标准色）

* 粒子特效颜色（需要高对比度）

* 奖状/庆祝动画颜色
