# 教学工具箱 - Electron桌面版

让课堂更有趣·让教学更高效

## 功能特点

- 🃏 **卡牌点名器** - 随机点名，带有精美的扑克牌翻转动画
- 🎲 **趣味挑战** - 刮刮乐形式的惩罚/挑战抽取
- 🎯 **大屏模式** - 适配老师投屏上课使用
- 🔊 **音效系统** - 丰富的交互音效
- 📱 **响应式设计** - 支持各种设备尺寸

## 安装运行

### 方式一：直接运行（开发模式）

1. 确保已安装Node.js（建议v18+）
2. 进入项目目录
3. 安装依赖：
   ```bash
   npm install
   ```
4. 运行：
   ```bash
   npm start
   ```

### 方式二：打包成安装包

1. 安装依赖：
   ```bash
   npm install
   ```
2. 打包Windows安装包：
   ```bash
   npm run build:win
   ```
3. 打包后的文件在 `dist` 文件夹中

## 与网页版的区别

| 特性 | 网页版 | Electron版 |
|------|--------|-----------|
| 运行环境 | 浏览器 | 独立窗口 |
| 离线使用 | 需要Service Worker | 完全原生离线 |
| 系统集成 | 有限 | 完全支持 |
| 性能 | 一般 | 更好 |
| 安装方式 | 浏览器打开 | 独立安装包 |

## 注意事项

- Electron版会自动禁用Service Worker（因为有更好的离线机制）
- manifest.json在Electron中被忽略
- 如需托盘功能，需要添加icon.png图标文件
